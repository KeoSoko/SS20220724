2️⃣ Auto-Generate Slug in handleSubmit

Inside handleSubmit, before calling mutation, generate slug.

Add this helper at top of file:

function generateSlug(label: string): string {
  return label
    .toLowerCase()
    .trim()
    .replace(/[^a-z0-9\s]/g, "")
    .replace(/\s+/g, "_")
    .replace(/_+/g, "_");
}
Modify handleSubmit

Replace:

createCategoryMutation.mutate(data);

With:

const slug = generateSlug(data.displayName);

createCategoryMutation.mutate({
  ...data,
  name: slug,
});
Modify edit logic as well

When editing:

Replace:

updateCategoryMutation.mutate({ id: editingCategory.id, data });

With:

const slug = generateSlug(data.displayName);

updateCategoryMutation.mutate({
  id: editingCategory.id,
  data: {
    ...data,
    name: slug,
  },
});
